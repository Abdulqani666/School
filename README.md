# Abubakar Sadiiq School Management System

## Deployment on Render

### 1. Push to GitHub
Upload this folder to a GitHub repo.

### 2. Create Web Service on Render
- Build Command: `npm install`
- Start Command: `node server/server.js`
- Environment: Node

### 3. Default Logins
- Superior Admin: `superadmin` / `super123`
- School Admin: `admin` / `admin123`
